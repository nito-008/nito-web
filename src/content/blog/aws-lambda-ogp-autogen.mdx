## Satori+resvgを使ってみる

[Satori](https://github.com/vercel/satori)というJSXをSVGにレンダーするライブラリと、[resvg](https://github.com/linebender/resvg)というSVGをPNGに変換するライブラリ（Rust実装）を組み合わせる方法が主流らしい。Vercelが提供しているOGP生成ライブラリである[@vercel/og](https://vercel.com/docs/og-image-generation)の実装もこうなっている。

```ts
const svg = await satori(<OgImage title={title} />, {
      width: 1200,
      height: 630,
      fonts: fontData,
    });
```
